Utilities (Subsystem of Planning Code)

The utilities system provides a set of basic functions, macros, and data types that are used throughout the other systems.
This system is copied from Artificial Intelligence Modern Approach (AIMA) and modified by Seiji Koide.

utilities\:
  • utilities.cl Basic utility functions and macros, used throughout the code.

File utilities\utilities.cl

Basic utility functions and macros, used throughout the code.

This file is copied from AIMA utilities/utilities.lisp by Seiji. Loop macro and related macros for control are removed and all of codes are revised for Common Lisp loop macros.

Control Flow Macros

deletef (item sequence &rest keys &environment env) [macro]

Destructively delete item from sequence, which must be SETF-able.

define-if-undefined (&rest definitions) [macro]

Use this to conditionally define functions, variables, or macros that may or may not be pre-defined in this Lisp. This can be used to provide CLtL2 compatibility for older Lisps.
